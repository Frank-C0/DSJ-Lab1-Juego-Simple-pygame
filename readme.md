# Juegos-Lab1

## Getting Started

1. Clone the repository:
    ```sh
    git clone https://github.com/Usuario/Juegos-Lab1.git
    ```
2. Navigate to the project directory:
    ```sh
    cd Juegos-Lab1
    ```
3. Run the main script:
    ```sh
    python main.py
    ```
